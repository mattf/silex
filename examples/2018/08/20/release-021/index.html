Silex 0.2.1

20 August 2018

Silex 0.2.1 is released!

Here are the main improvements and additions:

  • The SampleSink class is now generalized to support arbitrary numeric types with Fractional and NumericLimits instances.
  • Minor performance improvements and code cleanups in the self-organizing map implementation.
  • This will be the last release to support Scala 2.10.

You can:

  • Include Silex in your projects,
  • fork Silex on GitHub, and
  • read the API docs.
